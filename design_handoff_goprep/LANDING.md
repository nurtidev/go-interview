# Лендинг GoPrep — спека для реализации

Референс: `GoPrep Лендинг.dc.html` (открыть в браузере из этой папки). Стек и токены — те же, что в `README.md`; лендинг использует принятое направление 2c (графит, светлая «бумага») + дозированный liquid-glass.

## Структура страницы (сверху вниз)
1. **Навбар** — плавающая glass-пилюля (sticky, по центру, max-width 920px, height 52px): `background:rgba(255,255,255,.55)`, `backdrop-filter:blur(20px) saturate(1.4)`, border `rgba(38,36,31,.1)`, inset-подсветка сверху `rgba(255,255,255,.8)` + тень `0 8px 28px rgba(30,22,12,.08)`. Лого GoPrep_, ссылки (Как это работает / Программа / Отзывы / FAQ), Войти + primary CTA.
2. **Hero** — грид 1.05fr/.95fr. Слева: mono-оверлайн «ТРЕНАЖЁР СОБЕСЕДОВАНИЙ · SENIOR GOLANG», H1 PT Serif 50/1.15 «Оффер senior-грейда — это не удача. Это подготовка.», сабтекст, primary CTA + glass secondary «Попробовать вопрос ↓», 3 метрики (156 / 24 / 15 мин, mono 22). Справа: glass-карточка вопроса (лестница: middle раскрыт, senior за ссылкой, 4 кнопки самооценки) — `rgba(255,255,255,.72)`, blur 24, radius 18; поверх неё плавающая glass-пилюля таймера «18:42 · фокус 25/5» (absolute, правый низ). Фон hero: два больших radial-gradient «световых поля» (тёплое справа сверху `rgba(190,178,160,.28)`, холодное слева снизу `rgba(170,180,195,.22)`), overflow hidden.
3. **Полоса компаний** — «ГОТОВИМ К ФОРМАТАМ СЕКЦИЙ» + Яндекс/Ozon/Авито/Т-Банк/СБЕР/VK (текстом).
4. **Блок боли** (тёмный, фон `--ink` #26241F): заголовок «Вы пишете на Go пять лет. И всё равно плывёте на вопросе про закрытый канал», справа 3 пункта (01–03) с hairline `#3A3833`.
5. **Как это работает** — 3 строки-грида (64px номер / текст / карточка 380px): учебник (прогресс глав), лестница (бейджи middle✓/senior✓/deep… + самооценка), лайвкодинг (тёмная мини-консоль PASS/FAIL + таймер).
6. **Отзывы** (фон `#F5F3EE`): 3 glass-карточки (rgba(255,255,255,.75), blur 16, radius 14), цитата PT Serif italic, аватар-инициалы + должность. Тексты — плейсхолдеры, заменить на реальные.
7. **Программа** — слева заголовок «Только то, что спрашивают. Ничего „на всякий случай"» + кнопка, справа список 6 тем с количеством вопросов (hairline-строки).
8. **FAQ** — 4 вопроса (Чем лучше конспекта / Сколько времени / Я мидл — рано? / Это платно?), hairline-строки, max-width 760px.
9. **Финальный CTA** (тёмный `--ink`): «Ваше следующее собеседование уже назначено. Вопрос в том, кто к нему готов.» + светлая кнопка + «Без карты…». Футер внутри тёмного блока.

## Принципы
- Glass — только 4 места: навбар, hero-карточка, пилюля таймера, карточки отзывов. Остальное — плоская «бумага» с hairline.
- Все hex/шрифты — из токенов README (`--ink`, `--hairline`, семантические тинты). Радиусы стекла: 18 (карточка), 14 (отзывы), 999 (пилюли).
- fallback для `backdrop-filter`: непрозрачный `#fff`/`--surface` через `@supports not`.
- Тон копирайтинга — уверенно-продающий, но без агрессии; заголовки PT Serif, UI-тексты Plex Sans. Копирайтинг брать дословно.
- Мобайл: hero в одну колонку (карточка под текстом), навбар-пилюля сжимается (ссылки в бургер), метрики в ряд по 3, гриды секций в одну колонку. Без горизонтального скролла.
- Плашка компаний — текстовая; не использовать логотипы без прав.
