# content-private — приватный слой тренажёра

Личный материал для приватного инстанса GoPrep (тренажёр «для своих»).
Загружается ТОЛЬКО когда задан env `PRIVATE_CONTENT_DIR` (на публичном
инстансе не подключается). Репозиторий приватный.

НЕ подключать на публичном/продающемся инстансе: материал переработан из
сторонних источников для личного использования двух соавторов.
